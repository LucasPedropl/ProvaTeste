## Questão 1 — (3,0 pontos)

Considere o método abaixo, escrito na linguagem de programação Java, para contagem de elementos de um vetor iguais a um determinado número:

```java
public int contadorNumero( int[ ] vetor, int numero ) {
    int contador = 0;
    int indice = 0;
    if ( vetor.length > 0 ) {
        while ( indice < vetor.length ) {
            if ( vetor[indice] == numero ) {
                contador = contador;
            }
            indice = indice + 1;
        }
    }
    return contador;
}
```

Se não há defeitos, crie **3 casos de teste**. Se há defeito(s), crie **3 casos de teste** da seguinte forma:

1. Um que **não executa** um ou mais defeitos;
2. Um que **executa** um ou mais defeitos, mas **não causa falha**;
3. Um que **executa** um ou mais defeitos e **causa falha**.

Adicionalmente, se há defeito(s), faça a(s) correção(ões), indicando a linha ou trecho e a correção correspondente.

> [!NOTE]
> Não há a necessidade de utilizar um critério para a geração dos casos de teste.

## Questão 2 — (3,5 pontos)

Aplique um critério estrutural de teste no método abaixo para criar **3 casos de teste**. Especifique o nome do critério e mostre o passo a passo de sua aplicação.

```java
public boolean Primo( int numero ) {
    boolean verifica;
    int divisor = 1;
    int resultado;
    int numeroDivisoes = 0;
    if ( numero < 2 ) {
        verifica = false;
    }
    else {
        while ( divisor <= numero ) {
            resultado = numero % divisor;
            divisor++;
            if ( resultado == 0 ) {
                numeroDivisoes = numeroDivisoes + 1;
            }
        }
        if ( numeroDivisoes == 2 ) {
            verifica = true;
        }
        else {
            verifica = false;
        }
    }
    return verifica;
}
```

## Questão 3 — (3,5 pontos)

Aplique um critério, especificando e mostrando o passo a passo da aplicação, para criar **3 casos de teste** que devem ser usados no teste de um sistema que calcula o imposto de renda retido na fonte de um funcionário.

O sistema recebe o salário mensal bruto do funcionário e aplica uma alíquota conforme a **Tabela 1** para encontrar o valor do imposto e subtrair esse valor do salário bruto, informando o salário mensal líquido. O sistema recebe também a quantidade de filhos desse funcionário, um valor entre **0 e 5**, e para cada filho é aplicada uma dedução no imposto de **R$ 10,00**.

Os casos de teste devem contemplar **3 faixas salariais distintas** da tabela. Pode-se colocar a equação correspondente ao cálculo e não o resultado em cada caso de teste (por exemplo, colocar `2.500,00 * 0,05` em vez de `125,00`).

### Tabela 1: Faixas salariais e alíquotas

| Salário mensal | Alíquota |
| :--- | :---: |
| Até R$ 2.000,00 | 0% |
| De R$ 2.000,01 a R$ 3.000,00 | 5% |
| De R$ 3.000,01 a R$ 4.000,00 | 10% |
| De R$ 4.000,01 a R$ 5.000,00 | 15% |
| Superior a R$ 5.000,00 | 20% |
